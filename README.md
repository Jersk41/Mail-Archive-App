# Mail-Archive-App
Web Based Mail Archive Application with Native PHP and MySQL

## Technology Stack
- PHP 7
- Bootstrap 4

## Running The Application
1. Install XAMPP or Default Apache Server
2. Start Apache and MySQL server
3. Clone or download and extract this repository inside your `htdocs` folder
4. Create database with name `pengarsipan_surat.php`
5. Import Database `pengarsipan_surat.sql` inside `library` folder
6. Access web by typing in url <http://localhost/Mail-Archive-App/> or __your project directory__
7. And Done Register for creating the account first and login to the application
